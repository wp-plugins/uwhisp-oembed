== Description ==

This plugin allows you to copy and paste a uWhisp URL into a page or post and it will display a uWhisp player

== Installation ==

1. Upload `uwhisp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Paste a uWhisp URL where you want the player to show up.

== Frequently Asked Questions ==

= How do I get the player to show up? =

Paste a uWhisp URL where you want the player to show up.
